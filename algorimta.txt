PROGRAM CetakNilai

nilai 70-80 cetak nilai B
nilai 60 - 70 cetak nilai C
dibawah 50 cetak nilai D

	cetak A
if nilai > 70 then
	cetak B
if nilai > 60 then
	cetak C
else 
	cetak D
